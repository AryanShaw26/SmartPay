# SmartPay 💳

SmartPay is a modern fintech web application designed to provide a seamless digital payment experience. Inspired by platforms like Paytm, PhonePe, and Google Pay, it allows users to manage their digital wallets, send and receive money, scan QR codes, track transactions, analyze spending patterns, and generate transaction statements.

The application is built using a scalable full-stack architecture with React.js for the frontend, Django REST Framework for the backend, and PostgreSQL for data storage. SmartPay focuses on security, performance, and user experience while demonstrating real-world payment workflows and financial management features.

## Key Features

* Secure JWT Authentication
* Digital Wallet Management
* Send & Receive Money
* QR Code Payments
* Transaction History & Filters
* Expense Analytics Dashboard
* PDF Transaction Statements
* Email Notifications
* Beneficiary Management
* Responsive Fintech UI
* Dark Theme Dashboard
* Admin Management Panel

## Project Goal

The goal of SmartPay is to simulate a real-world digital payment ecosystem while showcasing full-stack development, secure transaction handling, REST API design, database management, and modern frontend development practices suitable for software engineering and placement interviews.
